Deals with Exceptions in lens spaces L(p,q), p <= 13.
It shows that
* 4_23 != mirror(5_1)
* 5_26 != 5_27
* 5_76 != mirror(5_76)

Requirements:
* SnapPy - http://www.math.uic.edu/t3m/SnapPy/
* SageMath - http://www.sagemath.org

Or upload to SageMath Cloud project:
https://cloud.sagemath.com/
and run .sagews files
